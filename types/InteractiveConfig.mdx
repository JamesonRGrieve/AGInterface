# Configuration

AGiXT Interactive is configurable in three ways: 
- Through environment variables
- Through the props of the AGiXTInteractive component
- Through search params (if enabled in the environment)

| **Configuration Option**    | **Option Description**                                                                            | **Default Value** | **Environment Variable** | **Component Prop**     | **Query Param** |
| --------------------------- | ------------------------------------------------------------------------------------------------- | ----------------- | ------------------------ | ---------------------- | --------------- |
| (Default) Conversation Name | Controls the conversation that is selected if there is no cookie. Use '-' for a new conversation. | -                 | AGIXT_CONVERSATION       | overrides.conversation | conversation    |
