# Configuration

AGiXT Interactive is configurable in three ways: 
- Through environment variables
- Through the props of the AGiXTInteractive component
- Through search params (if enabled in the environment)

## General Options

|    Configuration Option     |                                        Option Description                                         |      Default Value      |   Environment Variable    |           Component Prop           |    Query Param    |
| :-------------------------: | :-----------------------------------------------------------------------------------------------: | :---------------------: | :-----------------------: | :--------------------------------: | :---------------: |
| (Default) Conversation Name | Controls the conversation that is selected if there is no cookie. Use '-' for a new conversation. |           '-'           |    AGIXT_CONVERSATION     |       overrides.conversation       |   conversation    |
|      Conversation Mode      |                             Determines how conversations are managed                              |        'static'         |  AGIXT_CONVERSATION_MODE  |                 -                  |         -         |
|        AGiXT API Key        |                                         API key for AGiXT                                         |           ''            |       AGIXT_API_KEY       |                 -                  |         -         |
|        AGiXT Server         |                                       Server URL for AGiXT                                        | 'http://localhost:7437' |       AGIXT_SERVER        |                 -                  |         -         |
|         Auth Server         |                                     Authentication server URL                                     |           ''            |        AUTH_SERVER        |                 -                  |         -         |
|            Agent            |                                       Default agent to use                                        |       'gpt4free'        |        AGIXT_AGENT        |               agent                |       agent       |
|        Insight Agent        |                                      Agent used for insights                                      |            -            |    AGIXT_INSIGHT_AGENT    |     overrides.insightAgentName     |   insightAgent    |
|            Mode             |                           Interaction mode (prompt, chain, or command)                            |        'prompt'         |        AGIXT_MODE         |           overrides.mode           |       mode        |
|         Prompt Name         |                                     Name of the prompt to use                                     |        undefined        |     AGIXT_PROMPT_NAME     |          overrides.prompt          |      prompt       |

|           Command           |                                        Command to execute                                         |           ''            |       AGIXT_COMMAND       |         overrides.command          |      command      |
|  Command Message Argument   |                           Argument name for the message in command mode                           |        'message'        | AGIXT_COMMAND_MESSAGE_ARG |    overrides.commandMessageArg     | commandMessageArg |

|       Chain Arguments       |                                      Arguments for the chain                                      |           {}            |     AGIXT_CHAIN_ARGS      | overrides.chainRunConfig.chainArgs |     chainArgs     |
