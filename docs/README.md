## Installation

### Cloning

```sh
git clone git@github.com:JamesonRGrieve/AGInterface.git --recursive
```

**This repository uses submodules and the `--recursive` flag is important!**

### Requirements

- NodeJS >= 20

```sh
npm install
npm run dev
```

### Environment

```
APP_NAME=
APP_URI=http://localhost:1109
API_URI=http://localhost:1996
```
